## 23:25 — 趋势判定口径改为YoY同期对比

### 变更
- 销量趋势：从"前后半段均值对比"改为"26年AP1-6总和 vs 25年AP1-6总和"
  - 变化>±5% → 上升/下降，±5%内 → 平稳
  - 两年同期都无数据 → 数据不足
- 舆情趋势：从"前后半段均值差"改为"26年AP1-6均值 vs 25年AP1-6均值"
  - 差>0.1 → 改善，差<-0.1 → 恶化，±0.1内 → 稳定

### 新统计
- 销量趋势：下降207 / 上升115 / 平稳12 / 数据不足124
- 舆情趋势：稳定221 / 恶化151 / 改善86
- 数据不足124家原因：部分医院25年同期或26年同期无销量数据

### 依据
- 2026年只有AP1-AP6有实际销量数据（AP7-12完全空）
- 同期对比排除季节性干扰，比前后半段更合理
- 用户指定口径："26年数据总和小于25年同期数据→下降"

## 23:40 — 销量趋势改为5类口径（放宽数据缺失判定）

### 用户新规则
- "有数据"= 某年AP1-6窗口内销量合计>0（最多缺5个月、至少1个月有销量即算正常）
- 25无26有 → 上升（新进院增长）
- 25有26无 → 业务丢失（单独一列，停购/退出）
- 25无26无 → 数据缺失

### 销量趋势5类判定
- 25有+26有：26比25 >+5%上升 / <-5%下降 / ±5%内平稳
- 25无+26有 → 上升
- 25有+26无 → 业务丢失
- 25无+26无 → 数据缺失
